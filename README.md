📊 PhonePe Transaction Analysis Dashboard

The PhonePe Analysis Dashboard is a data visualization project that I built using Power BI to analyze digital payment trends and user behavior across India. Through this project, I worked on transforming raw CSV data into interactive dashboards that make complex transaction data easier to understand and explore.

My goal with this project was to uncover useful insights related to transaction patterns, user growth, regional performance, and overall digital payment activity in a simple and visually engaging way.

🎯 Project Overview

In this project, I analyzed PhonePe transaction and user datasets to understand how digital payments are growing across different regions and time periods.

Using Power BI, I created dashboards that help visualize:

Transaction amount and count over time
State-wise and district-wise transaction activity
Quarterly transaction trends
Registered users and app opens
Transaction type distribution
Regional and device-based user insights

Through this analysis, I aimed to convert raw data into meaningful business insights that could help in trend analysis and decision-making.

🔑 Key Features
📌 Total Transactions Overview

I created a section that allows users to explore the total transaction amount and transaction count across different years and quarters.

🗺 State & District Analysis

I used location-based visuals to compare transaction activity across states and districts, making it easier to identify high-performing regions.

🎛 Interactive Filters

To make the dashboard more user-friendly, I added interactive filters for year, quarter, and region so the visuals update dynamically based on selections.

💳 Transaction Type Breakdown

I included a breakdown of different transaction categories such as:

Peer-to-peer payments
Merchant payments
Recharges
Financial services
Other transaction types

This helped me better understand how users interact with different types of digital transactions.

📍 Regional Insights

I highlighted top-performing states and regions to identify where PhonePe usage and transaction contribution are strongest.

📱 Device Usage Analysis

I also explored device and mobile brand usage patterns to understand user distribution from a technology perspective.

🛠 Tools & Technologies I Used
Power BI – to build interactive dashboards and visual reports
CSV Files – as the primary source of raw data
Power Query – for cleaning, transforming, and shaping the data
